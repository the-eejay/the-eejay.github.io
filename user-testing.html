User Evaluation Report

The primary goal of this user evaluation test is to find out the accessibility of the game and usability of the game. To get the best results, the test requires several users to trial it, and was conducted with 5 users. They all have different experience with playing video games. In that way, the results will be fair to everyone.

Users:

Name Gender Age Experience playing video games Impairment
Alex Male 24 Always None
Jack Male 23 Often None
Tim Male 22 Some None
Andy Male 22 Often None
Lina Female 23 None None

Test Methods

Compatibility

Q. Did the game run smoothly enough on your computer?


Name Alex Jack Tim Andy Lina
Scale 0 0 2 1 2

Alex – As a gamer, I have a high performance computer at home. So, I could run the game smoothly although, the graphic setting was maximum.

Jack – I am using under 1000 dollars’ worth laptop. I have no problem with running the game, but the computer starts to get hotter the longer I play the game.

Tim – I can run the game when the graphic setting is medium. If any higher than that, the game starts to get pixelated.

Andy – The game can be run smoothly, only in condition where the graphic setting is medium.

Lina – Perhaps due to my computer’s compatibility, the game is not running smoothly in highest graphic settings. It also lags a bit when the enemies spawn around me.

Average Points – 1

User feedback on compatibility (Recommendations):

  • Average computers can run the game smoothly when not on the highest graphical setting
  • It is suggested that the product should mention that medium graphic setting is highly recommend in the tutorials or in the setting up guide

Observations:

This test was designed to see whether average computers run the game smoothly or not. Therefore, consumers do not have to worry about if their computer can handle the game. Three of the users seemed to be worried when the graphic setting reached maximum. However, in most of game time, the game was running smoothly.

Functionality of game

Q. Did you find that features in the game were easy to use/control?


Name Alex Jack Tim Andy Lina
Scale 1 1 2 2 2

Alex – Most features were easy to use. I used all features except for the upgrade feature. I was busy with playing game, and forgot it was there a lot of the time.

Jack – All features were intuitive, so I was able to play the game without any tutorials.

Tim – I could not see health bar of my ship properly at the beginning of the game, due to location of the bar.

Andy – I had no problem with controlling ships and shooting bullets. However, I did not know how to change space ships until someone told me.

Lina – In the perspective of non-gamer view, all features were fairly intuitive, but I think players will have no problem with using features if they read the tutorials.

Average Points – 1.6

User feedback on functionality of the game (Recommendations):

  • The health bar should be moved to top, so it becomes more visible
  • Upgrade can be removed. Too busy with playing
  • Add tutorial mode for beginners

Observations:

This test was designed to see whether players can use functionalities in the game or not. Most of the users seemed to use most features in the game. Upgrade feature seemed to not be used much during the game, because all users were busy with shooting enemies. Moreover, some of the users did not see the health bar during the game. However, they became aware of all the features and used them after being told during testing.

Navigation (UX design)

Q. Was the game easy to use? Are you ever lost when using it?


Name Alex Jack Tim Andy Lina
Scale 1 0 0 0 0

Alex – How can I go back to main menu when I play a single player mode? Do I have to die to get the main menu always? Other than this, the game was easy to use; I never got lost during it.

Jack – I found no problem when I’m playing the game.

Tim – I found it pretty straight forward.

Andy – It seemed relatively easy enough. I didn’t get lost when trying to do something.

Lina – I found it quite simple and intuitive.

Average Points – 0.2

User feedback on navigation (Recommendations):

  • It is suggested that pause button or exit button should be created in the game. In that way, players can exit or pause any time they want

Observations:

This test was designed to see whether users correctly use features and see the UX design is fluently working or not. All users had no problem with using the game, but only one person seemed to be frustrated, because he chose a wrong ship and start the game. Other than this, this test was successfully finished.

Sounds

Q. Do you think that the sound effects were appropriate?


Name Alex Jack Tim Andy Lina
Scale 2 1 0 1 1

Alex – Most sound effects were fairly well designed. Maybe change the background music when you are in the game mode. It kind of reminded me arcade games like Tetris.

Jack – Sounds were amazing, but too loud for me, even when the volume was low.

Tim – All sounds were good to me.

Andy – Great sounds, but I was annoyed by the sounds.

Lina – I found that there was no music in the main menu. Where is it?

Average Points – 1

User feedback on sounds (Recommendations):

  • Add background music in the main menu
  • Add explosion sound effects e.g. when enemies die

Observations:

This test was designed to see whether sound effects are appropriate for all users or not. For example, does it remind you of a space game? The end results were finished positively. One of users took off the headphones after the first time playing. Moreover, other users check the volume of the sound in the main menu. The users heard the rest of sound effects and had no problem with them.

Summary

This user evaluation report has documented the hidden discoveries from the following methods.

User evaluation test has been examined by four different methods:

  • Compatibility
  • Functionality of game
  • Navigation (UX design)
  • Sounds

According to these methods, the feedback has been fairly positive. Furthermore, the product has benefited significantly from these tests and recommendations from testers. Since all users/testers had different experience with playing game, we were able to receive various opinions and feedback suggestions from them. As a result, the game now covers a variety of people.
