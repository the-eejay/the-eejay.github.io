Startup Guide

Installation

  • Open http://the-eejay.github.io/ in your web browser of choice
  • Download the Windows/Mac/Linux version onto your computer
  • Extract the zip file and open SpaceCOG.exe

Controls

  • W, A, S, D keys control the up, left, down and right movement respectively.
    • Ships move relative to the directional key the player presses and not the direction that the ship faces.
  • Aiming the ship is controlled via the mouse.
    • The ship will face in the direction the mouse is aimed.
  • Holding the left mouse button triggers weapon fire.

Gameplay and Objective

The objective of the game is to survive and destroy the 20 waves of enemies that spawn randomly around the players. The players can earn points together to reach a high score, however; a player will stop receiving points after they die.

Singleplayer: How to start

  • Open SpaceCOG client
  • Select a ship at the main menu screen
  • Press “Singleplayer”
  • The game will start momentarily

Multiplayer: How to host

  • Open SpaceCOG client
  • Select a ship at the main menu screen
  • Press “Host”
  • Input a game name and press “Create Game”
  • Wait for players to join
    • Once the room is full, the game will start automatically
    • Else, the host can press “Play” to start with fewer players

Multiplayer: How to join

  • Open SpaceCOG client
  • Select a ship at the main menu screen
  • Press “Join”
  • Select a room from the list that loads
  • After joining a room, wait for the host to start the game
